# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-18
### Added
- Initial release of `rust-cp-toolkit`.
- `init` command to bootstrap projects.
- `compile` and `run` commands for executing solutions.
- `test` command for running against sample data.
- `stress` command for finding edge cases.
- `benchmark` command for timing execution.
- `compare` command for evaluating outputs.
- `clean` command for removing temporary artifacts.
